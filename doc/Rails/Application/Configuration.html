    class Rails::Application::Configuration

    Add config.vbulletin.<parameter> accessor. Supported parameters are:

    • config.vbulletin.cookie_salt = 'COOKIE_SALT' - required if you want to user "Remember me" functionality. It contains a VBulletin COOKIE_SALT special value, which is used when creating "Remember me" cookie. Cookie salt is located in includes/functions.php in line 34 of your VBulletin package and is unique for every application.
